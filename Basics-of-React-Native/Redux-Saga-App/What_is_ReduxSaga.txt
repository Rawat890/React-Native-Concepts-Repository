What is middleWare ? 
Suggested way to use with redux with the custom functionality
Provides a point between the 2 points that is dispatch action to the moment it reaches the reducer
Generally used to perform the async operations
So, middleware is a layer of code that runs between dispatching an action and the reducer executing it. It allows you to extend Redux's functionality by intercepting, modifying, or enhancing actions. 

What is Redux saga ?
It is a middleware that provides the redux store a way to interact with the api and perform the async operations
It is asynchronous and better error handling than the redux Thunk
Uses the generator functions that has the ability to pause and resume the execution of function and perfect fit for async operations
