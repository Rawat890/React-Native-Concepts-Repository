Formik is a library that helps you deal with forms with react.
Why ? 
Managing the form data
Form submission
Form validation and displaying error messages

Allows us to deal with forms in a scalable and performant manner

Course structure -
Build a simple Form
use Formik hook
Manage the form state
handle form submission
validation
Formik components
Reusable components for input , text area, select, radio buttons
Build a use registration form
Wire a ui library
